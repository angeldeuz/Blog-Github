fetch y push

traer cosas del repositorio de github con fetch y mandarlas al repositorio con push "git push origin master"

Cuando creamos un repositorio en github o cualquier otro lado y tenemos archivos nuevos como por ejemplo el redme hay que traer esos cambios primeros a mi carpeta local en git para asi poder dar push a todo nuestro proyecto funcional igual para actualizarlo hay que tener los ultimos cambios del repositorio remoto
